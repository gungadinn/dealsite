# 중고거래 게시판

* 프로젝트명 : 중고거래 게시판
* 개발기간 : 2018.05.27~2018.06.15 (3주/3명)
* 개발환경
  * OS : Mac OS / Window 10
  * IDE : Eclipse
  * Language : jsp, html, ..

* 프로젝트 목적
  - 사용자들의 물건 거래를 위한 사이트로 사용자들이 구매 게시물, 판매 게시물을 작성할 수 있고 거래가 가능한 웹 페이지

<br>

### 기능

---

* 회원가입, 로그인, 로그아웃, 회원정보수정
* 구매/판매
* 내 글 목록 보기
* 쪽지 주고받기
